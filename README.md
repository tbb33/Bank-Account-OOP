# Bank-Account-OOP

OOP shown in this app: classes, obj instances, instance vars, class vars, doc strings, inheritance, sub classes, attributes, instantiation, constructor, class methods, data members
 
Execute "acct.py" to run this program.

This file shows OOP in a basic bank account program that can withdraw, depost, and transfer funds from the balance.
There are 2 object instances (2 checking bank accounts - jill_checking and jack_checking) and their 2 respective balances are in a .txt file.
Note: balance.txt was used early in building the program as seen in history.
